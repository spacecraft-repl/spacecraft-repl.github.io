What is SpaceCraft?

SpaceCraft is an open source REPL (Read-Eval-Print-Loop) that enables real-time collaboration between users. It enables pair-programming without any additional setup, by providing a platform for users to execute code in the browser for Ruby, JavaScript, and Python.
